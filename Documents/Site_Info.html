Project Description

This website is a nostalgic journey back to the 1990s, recreating the authentic Windows 95/98 desktop experience in a modern web browser. The project combines retro aesthetics with contemporary web technologies to deliver an immersive desktop environment that feels like stepping into a time machine.

The site features a complete Windows 95-style desktop interface with draggable windows, a functional taskbar, desktop icons, and a Start Menu. Users can interact with various applications including a web browser, music maker, document viewer, and more - all styled to match the classic Windows 95 look and feel.

Technology Stack

The project is built using modern web technologies:

  • React - Modern JavaScript library for building user interfaces
  • Vite - Fast build tool and development server
  • React95 - Windows 95 UI component library
  • Express.js - Backend server for API proxying
  • Discogs API - Integration for vinyl collection data
  • CSS3 - Custom styling with animations and retro effects
  • HTML5 - Modern markup with semantic structure

Key Features

Windows 95 Desktop Interface
Complete desktop environment with draggable windows, taskbar, Start Menu, and desktop icons. All interactions mimic the classic Windows 95 experience.
Web Browser Application
Custom browser interface styled like Internet Explorer, featuring tabbed browsing and integration with websites you can browse.
Music Maker
Interactive music creation tool with drum sequencer and synthesizer, complete with mixer controls and BPM adjustment.
SEO Checker Tool
Website analysis tool that examines SEO metrics, meta tags, and provides comprehensive reports on any given URL.
Document Viewer
Built-in PDF viewer for displaying documents and CV within the Windows 95 interface.
